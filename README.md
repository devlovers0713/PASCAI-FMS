# 파일 정리 자동화

## 목표
- 정리되지 않은 파일을 AI로 분류·이동해 폴더 구조를 정돈.
- 분류 기준이 없으면 새 디렉터리를 만들어 배치.
- 판단이 어려운 파일은 사용자 검토용으로 남겨 안전하게 처리.

## 기본 동작
- 파일 내용과 메타데이터를 분석해 규칙에 맞게 이동.
- 분류 기준에 맞는 디렉터리가 없으면 새 디렉터리를 생성해 이동.
- 중복 파일 처리:
  - 날짜와 내용이 모두 다르면 이름 끝에 -n을 붙여 구분.
  - 날짜만 다르고 내용이 같으면 한 개만 남기고 정렬에 반영.
  - 날짜가 같고 내용이 다르면 최신 시각본을 남기고 나머지는 삭제.
- 파일명과 내용이 어긋나면 내용에 맞게 이름을 수정.

## 분류 우선순위
- 자료 성격 파악: 학교 파일, 개인 프로젝트, 특정 기업 관련 자료 등.
- 학교 파일: 학년별 정렬.
- 개인 프로젝트: 프로젝트 단위 정렬.
- 기타 자료: 출처·주제를 기준으로 연관성 판단 후 정렬.
- 작업 날짜별 분류(주 단위).
- 분야별 분류: H/W, S/W, 통신, 제어, 회로 등.
- 문서 형식별 분류: pdf, dwg, dxf, docx, xlsx 등은 형식별 폴더로 묶고, 나머지는 연관성을 파악해 이름을 붙여 함께 보관.

## 리소스 사용 정책
- 노트북이 켜져 있으면 남는 리소스로 무거운 파일 정리·이관 연산 수행.
- 노트북이 꺼져 있으면 가벼운 파일만 정리·이관 연산 수행.
- 파일의 경중 판단은 노트북이 꺼져 있을 때 idle 작업으로 미리 계산.
- 프로그램이 리소스 과부하를 일으키지 않도록 제한.

## AI 처리
- 파일 제목만 볼지, 본문 내용까지 읽을지 결정 필요(현재 15자 이내 요약 가정).
- VectorAPI로 벡터화 후 Weaviate 등 Vector DB에 업로드해 클러스터 기반 검색/분류에 활용.
